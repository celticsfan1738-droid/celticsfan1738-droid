## 🛠️ Projects

### 🚇 [NYC Subway Arrival Display](https://github.com/charlie-ehn/nyc-subway-arrival-display)
A real-time subway arrival board built on an ESP32 + 3.5" TFT display. Pulls live data directly from the MTA's GTFS-RT API — no app, no phone, just plug it in. Fully configurable for any NYC subway station and up to 4 train lines.

> `C++` `ESP32` `Arduino` `MTA GTFS-RT API` `TFT_eSPI`
